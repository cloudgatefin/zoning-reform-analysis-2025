# PWA Screenshots

This directory should contain screenshots for the PWA app store listing:

## Required Screenshots

1. **Mobile (narrow)**: 540x720 pixels
   - Screenshot of the dashboard on a mobile device
   - Shows the hamburger menu, filters, and charts

2. **Desktop (wide)**: 1280x720 pixels
   - Screenshot of the full dashboard layout
   - Shows all components and visualizations

## How to Generate

1. Open the dashboard in a browser
2. Use browser dev tools to set the viewport size
3. Take a screenshot (browser screenshot or OS screenshot tool)
4. Save as PNG with the correct dimensions
5. Name them: screenshot1.png (mobile) and screenshot2.png (desktop)

## Tips

- Use a clean data state for screenshots
- Ensure all UI elements are visible
- Avoid showing placeholder or error states
- Use representative data that shows the dashboard's value
